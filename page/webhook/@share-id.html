Webhook is work-in-progress

Webhook support is planned for api.horse, but it is not implemented yet.

API horse is a side project, so new features are built as time allows.

What is the webhook feature?

The webhook tool is the opposite of the request creator. Instead of composing outbound requests, it will give you an endpoint you can send requests to so you can inspect the incoming payload.

It should make it easier to debug integrations, test third-party webhook providers, and share captured payloads when you need to show someone what happened.
